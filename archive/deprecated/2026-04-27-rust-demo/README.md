# Deprecated Rust Demo Archive

Date: 2026-04-27

This directory preserves the first WhaleCode Rust demo implementation after the
project direction moved from a from-scratch runtime to a Codex CLI upstream
substrate plus Whale overlay architecture.

Archived contents:

- `crates/`: the previous `whalecode-*` demo crates.
- `Cargo.toml`, `Cargo.lock`, and `rust-toolchain.toml`: the previous demo
  workspace metadata.
- `scripts/test-after-change.sh`: the previous demo smoke/regression script.

This archive is not part of the active product path. Do not continue feature
development here. If a detail is still useful, port it deliberately into the new
Codex bridge or Whale overlay layer with a migration note.
